# 🏥 Hospital Data Analysis Dashboard

## 📌 Project Overview

The **Hospital Data Analysis Dashboard** is an Excel-based data analytics project that transforms raw hospital data into meaningful insights using interactive dashboards and visualizations.

This project focuses on analyzing patient information, treatment costs, payment methods, insurance details, recovery status, and monthly admission trends to help understand hospital performance and patient patterns.

## 🎯 Project Objectives

- Analyze total hospital revenue generated.
- Understand patient distribution based on medical cases.
- Compare treatment costs by different payment modes.
- Analyze insurance coverage among patients.
- Track monthly patient admission trends.
- Identify key healthcare insights through visual dashboards.

## 📊 Dashboard Features

### 1. Treatment Cost Analysis
- Visualizes the total treatment cost based on payment methods:
  - Card
  - Cash
  - Insurance Claim
  - UPI

### 2. Patient Case Distribution
Shows patient percentage distribution among different medical cases:
- Cardiac
- Diabetes
- Cancer
- Other Cases

### 3. Hospital Revenue KPI
- Displays the total revenue generated by the hospital.

### 4. Average Patient Age
- Calculates the average age of admitted patients.

### 5. Insurance Analysis
Provides insights into:
- Total number of patients
- Patients with insurance
- Patients without insurance

### 6. Monthly Admission Trend
Tracks patient admissions from:
- January to December

### 7. Patient Recovery Analysis
Displays the total number of recovered patients.
